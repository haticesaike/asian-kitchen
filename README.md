# Asian Kitchen Menu 
Bu web uygulaması, restoranın menüsünü JSON verilerini kullanarak dinamik bir şekilde gösteren bir uygulamadır. JSON verileri kategoriye göre filtrelenir ve arayüzde gösterilir.

## Kullanılan Teknolojiler
* HTML
* CSS
* Bootstrap
* Javascript
